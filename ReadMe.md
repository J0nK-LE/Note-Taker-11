## About The Project
This app takes your notes and saves them in a database until they are ready to be deleted. Using express.js on the backend to make the whole thing work.

## Usage
Write your notes in the indicated fields then hit the save button on the top right corner. Click the title of the notes you'd like to open. If you'd like to add a new note, click the plus sign in the top right.
